# Meet

###### Last Updated 20210409

​

1.  Project Description:

        - Meet App is a PWA that displays events from GoogleCalendarAPI

    <br/> ​

        - This app allows users to:

          - Filter events by city.
          - Show/hide event details.
          - Specify number of events.
          - Use the app when offline.
          - Add an app shortcut to the home screen.
          - View a chart showing the number of upcoming events by city.

          ​

        - My role for this project:
          ​

          - Fullstack Developer: AWS Serverless + ReactJS PWA

            - Frontend: ReactJS + TailwindCSS
            - Backend: AWS Serverless + Google CalendarAPI

          - Tester

            - TDD Framework: Jest
            - BDD Framework: Cucumber
            - Gherkin
            - Atatus

        ​

2.  How to get the project running:

    - Running Test: Jest & Cucumber

      - Run `npm run test`
        ​

    - Running Project

           1. Run `npm run start`
           2. The project will be served at: `localhost:3000/`
